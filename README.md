# MCP Server - Binaries

Pre-built binaries for the MCP Server.

**Latest Version: v1.0.1

## Quick Start

### Linux (x86_64)
```bash
curl -L https://github.com/yourname/mcp-server-releases/releases/download/v1.0.0/mcp-server-linux-amd64 -o mcp-server
chmod +x mcp-server
./mcp-server
```

### macOS (Intel)
```bash
curl -L https://github.com/yourname/mcp-server-releases/releases/download/v1.0.0/mcp-server-darwin-amd64 -o mcp-server
chmod +x mcp-server

```
